# Conect Boi

Plataforma digital para compra e venda de gado no Brasil.

## Sobre o projeto

A Conect Boi foi criada como um projeto acadêmico com o objetivo de explorar soluções digitais para o mercado pecuário brasileiro.

O projeto propõe um marketplace que conecta:

- Produtores
- Compradores
- Corretores

Inspirado em modelos como o QuintoAndar, a plataforma busca aumentar a liquidez do mercado e ampliar o alcance das negociações.

## Funcionalidades do MVP

- Busca de lotes por região
- Visualização de ofertas disponíveis
- Página de detalhes do lote
- Início de negociação
- Página institucional explicando o modelo de negócio

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS

## Objetivo

Este projeto foi desenvolvido como um MVP para demonstrar o conceito da plataforma e explorar o potencial de digitalização do mercado de gado no Brasil.
